Startup scripts for STM32
=========================

All startup scripts are taken from the STM32 Standard Peripheral Library, or from 
code generated by the STM Cube (part of the CMSIS driver folder).

The scripts provided here are the "arm" variant, meant for ARM assembler or ARM-CC.
